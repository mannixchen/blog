# 详解

## $parent

```js
// 有个巧妙的 while 应用,
// 判断父组件是否为抽象, 并且还存在父组件, 继续向上寻找
while(parent.$options.abstract && parent.$parent) {
  parent = parent.$parent // 巧妙, 只要新的 parent 继续符合要求, 就会继续向上寻找, 直到符合不是抽象组件, 或者不在有上层父组件为止
}


// 此时再将 vm 的$parent 赋值为 parent
```

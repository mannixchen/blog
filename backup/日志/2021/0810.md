# 学习了

1. 捕获的引用
   1. 对捕获的引用有两种方式: 
      1. 自身匹配
      2. 字符串替换
   2. `\1` 指向表达式中的第一个捕获, 使用第一个捕获的内容配对标志
2. **字符串的replace方法**
   1. 在替换时候 可以使用 **`$1, $2` ** 等标记捕获序号 p272
   2. 分组和捕获都可用圆括号表示, 处理器会将所有的圆括号同时当做分组和捕获
3. 当不想圆括号代表捕获时候
   1. 在起始圆括号之后使用符号 `?:`
   2. 这就叫做被动子表达式, 此时只能被当做分组, 不能当做捕获
   3. 这样表达式引擎就可以不记忆和返回捕获结果, 减少很多工作
   4. 这样写付出的代价就是更加神秘 **难懂**
4. **replace 方法**
   1. replace 第一个参数是正则表达式
   2. 第二个参数可以是要替换的值, 也可以是一个函数
   3. 如果是函数, 就会对每一个匹配到的值都调用一遍, 用函数的返回值作为替换值
   4. 参数, 第一个参数就是匹配到的字符串, 第二个就是第一个捕获到的值
5. 匹配还可以做匹配 Unicode 字符 和 匹配转义字符的工作, 目前不好理解, 先不看
